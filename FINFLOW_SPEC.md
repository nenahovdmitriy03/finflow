# 💸 FinFlow — Требования к мобильному приложению

> Полная спека (исходник). См. README для быстрого обзора.

## Архитектура
Kotlin · Jetpack Compose · MVVM + Clean Architecture · Hilt · Room · Navigation Compose · DataStore · Vico (charts) · Lottie

## Палитра
- Primary: `#6C63FF`
- Income: `#2ECC71`
- Expense: `#FF6B6B`
- Light BG: `#F8F9FE` / Dark BG: `#1A1A2E`

## Экраны (MVP)
1. **Dashboard** — карточка баланса, FAB, последние транзакции, мини-цели, бюджет-индикатор
2. **Add Transaction** — сумма (с калькулятором), тип, категория, дата, кошелёк, заметка, фото чека
3. **Analytics** — bar / donut / line charts, экспорт PDF/CSV
4. **Goals** — прогресс-бар, расчёт «откладывай X ₽/мес», Lottie-конфетти при 100%
5. **Wallets** — наличные / карта / накопительный / крипто, переводы
6. **Budgets** — лимиты по категориям, push при 80% / 100%
7. **Profile** — валюта, тема, язык, бэкап, PIN/биометрия, кастомные категории

## Push-уведомления
| Триггер | Сообщение |
|---|---|
| Ежедневный ввод | «Не забудь записать расходы за сегодня 📝» |
| Превышение бюджета | «Бюджет на Еду исчерпан на 100%! 🚨» |
| Прогресс цели | «Ты на 50% пути к цели "Отпуск"! 🎯» |
| Достижение цели | «Цель достигнута! 🎉» |

## Безопасность
SQLCipher (опц.) · Биометрия + PIN · Автоблокировка · Локально по умолчанию

## Доступность
TalkBack · Масштабирование шрифта · Touch ≥ 48dp · WCAG AA

## Roadmap
- **v1.0 (MVP):** транзакции, категории, dashboard, базовая аналитика, цели, тёмная/светлая тема, Room
- **v2.0:** кошельки, бюджеты, экспорт, облако, повторы, виджет
- **v3.0:** мультивалютность, Open Banking, AI-советник, семейный аккаунт, Wear OS

## Нефункциональные требования
- Запуск < 2 сек, 60 fps
- Min API 26 (Android 8.0) · Target API 35 (Android 15)
- APK < 25 МБ · Полная работа оффлайн
- Unit-тесты ≥ 70%
